# Nicxonsolutions HTML

![Nicxonsolutions HTML](https://nicxonsolutions.com/logo.png)

This package is a fork of [LaravelCollective/html](https://github.com/LaravelCollective/html). Since the original package is no longer maintained, we have created this fork to provide continued support and compatibility for modern Laravel versions, up to Laravel 13.

## Installation

You can install this package via composer:

```bash
composer require nicxonsolutions/html
```

## Documentation

The original documentation for Forms & Html can be found at the [LaravelCollective](https://laravelcollective.com/docs) website. Most features remain the same, but under the `Nicxonsolutions\Html` namespace.

## Support

- Laravel 6.x - 13.x
- PHP 8.2+
